# Sistema-de-recomendaci-n-de-canciones-con-RL
presente proyecto establece el diseño de un agente inteligente en la recomendación musical basado tanto en un modelo de filtrado basado en contenido como en un proceso de aprendizaje por refuerzo, donde el sistema aprende de las recomendaciones en función de la retroalimentación que recibe de los usuarios.

La base de datos que se importa directamente es fma small
Todos los bloques de codigo tienen un titulo donde se indica su proposito así como las librerias que se usan y demás
